# Contributing to Open Panel

Thank you for your interest in contributing!

## Development Setup

1. Clone the repository
2. Install Go 1.22+ and Node.js 18+
3. Start backend: `cd backend && go run ./cmd/server`
4. Start frontend: `cd frontend && npm install && npm run dev`

## Pull Request Guidelines

- Keep changes focused and well-scoped
- Follow existing code style and conventions
- Update README and [docs/](docs/README.md) if adding user-facing features
- Test on Linux when changing system integration code

## Reporting Issues

Please include:
- OS and version
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs
