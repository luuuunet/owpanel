## 1. 产品概述

OWPanel 是开源 Linux 服务器运维管理面板，提供 Web 界面统一管理：

- 网站、SSL、CDN 缓存、WordPress / Node.js / Java 运行环境
- MySQL / PostgreSQL / Redis 及备份
- Docker 容器与 Compose 一键应用
- 防火墙、WAF、安全检测
- SSH 终端与 PAM（特权访问管理）
- 计划任务、备份、监控、集群
- 子账户按模块授权

**技术栈**：Go 后端 + Vue 3 前端，单二进制部署，内嵌静态资源。

## 2. 登录与界面

### 2.1 访问地址

| 场景 | 地址 |
|------|------|
| 默认 | `http://服务器IP:8888` |
| 启用安全入口 | `http://IP:8888/<安全路径>/` |

安全入口在 **面板设置 → 安全入口** 配置，可防止未授权扫描直接访问登录页。

### 2.2 首次登录

- 用户名：`admin`
- 密码：安装时随机生成，保存在 `{dataDir}/INITIAL_CREDENTIALS.txt` 或服务日志
- 若检测到弱密码/默认密码，会强制跳转修改密码

### 2.3 双因素认证（2FA）

在 **面板设置 → 安全** 中可启用 TOTP 双因素认证。启用后登录需额外输入 6 位动态验证码。

### 2.4 界面语言

- **登录页**：右下角语言按钮（简 / 繁 / EN）
- **登录后**：**面板设置 → 界面语言**，选择后立即生效并保存在浏览器本地
- 首次访问会根据浏览器/系统语言自动选择

### 2.5 主题

登录后右下角可切换浅色 / 深色 / 跟随系统；深色模式下可选多种配色变体。

## 3. 仪表盘

路径：**概览 → 仪表盘**

### 3.1 系统状态

- CPU、内存、磁盘、网络实时使用率
- 负载、磁盘 IO、网络 IO 趋势图
- 系统信息：主机名、系统版本、运行时间

### 3.2 健康评分

综合 CPU、内存、磁盘、负载、服务状态等因素给出健康分与建议项，便于快速发现风险。

### 3.3 一键优化

根据当前负载、内存与磁盘状态自动执行安全优化，例如：

- 释放页面缓存
- 刷新 Swap
- 清理 Docker 悬空资源、日志、包管理器缓存等

执行前会弹出确认，完成后展示逐步结果摘要。

### 3.4 释放内存

单独清理 Linux 页面缓存以释放内存（仅 Linux 有效，可能短暂影响磁盘性能）。

### 3.5 流量地图

世界地图展示访问来源国家/地区分布，支持：

- 按时间范围筛选
- 点击国家钻取：查看该国访问的域名、路径、来源、IP
- 配置地理策略：对指定国家/地区 **阻断** 或 **301 跳转**（需选择站点后应用）

### 9.4 自动化运维

## 11. SSH / PAM

路径：**SSH / PAM**（菜单统一入口，原独立「堡垒机」已合并）

页面顶部两个标签：**SSH 终端** | **PAM 管理**

### 11.1 SSH 终端

- 多标签 Web SSH，同时连接多台主机
- 快速连接：本机、集群节点、已授权 PAM 资产
- 密码 / SSH 密钥认证，密钥在终端内管理
- 收藏常用连接
- 内置 AI 助手：根据上下文建议命令（需在面板设置配置 AI）
- 快捷键：`Ctrl+Shift+T` 新标签、`Ctrl+Shift+W` 关闭、`F11` 全屏

### 11.2 PAM 管理

| 标签 | 功能 |
|------|------|
| **资产** | SSH 主机登记、分组、从集群导入、Known Hosts 指纹管理 |
| **访问申请** | JIT 临时访问：提交理由与时长，管理员审批 |
| **权限** | 用户 ↔ 资产授权（connect / sftp / readonly）、命令策略 |
| **会话审计** | 历史会话回放、日志下载、命令提取 |
| **在线会话** | 查看并强制断开活跃连接 |
| **账号管理** | 特权账号、自动轮换、推送、Vault 导入导出、发现账号 |
| **运维中心** |  adhoc 批量命令、作业模板、定时作业 |

从 PAM 资产点击「连接」会自动切换到 SSH 终端并建立会话；会话全程审计录像（管理员可回放）。

> 子账户需被授予 **SSH / PAM** 模块权限方可进入本菜单（仅连接被授权资产，管理功能多为管理员）。

## 18. 数据目录与 CLI

| 路径 | 用途 |
|------|------|
| `wwwroot/` | 网站根目录 |
| `server/{app}/` | 面板安装的软件 |

| `backup/` | 本地备份 |

### 18.2 CLI `op`

```bash
op              # 交互菜单
op info         # 面板 URL、端口、数据目录
op config       # 修改端口、安全入口、SSL
op restart      # 重启面板服务
op help         # 完整命令列表
```

### 18.3 远程部署（开发者）

```bash
cp scripts/deploy.env.example scripts/deploy.env
# 编辑 DEPLOY_HOST、DEPLOY_KEY 等
pip install paramiko
bash scripts/auto-setup.sh deploy   # 或 deploy-web.py / deploy-bin.py
```

详见项目根目录 [README.md](../../README.md)。

## 19. 常见问题

### 登录页 404 或无法访问

检查是否启用了 **安全入口**，URL 需包含正确路径前缀，例如 `http://IP:8888/你的安全路径/`。

### 子账户看不到 SSH 菜单

在用户管理中为该子账户勾选 **SSH / PAM** 权限。

### 计划任务不执行

1. 确认任务已启用  
2. 查看 **计划任务 → 状态** 与执行日志  
3. Linux 上确认系统 cron 或面板调度服务正常  

### 备份失败

检查目标路径权限、远程 FTP/SFTP/OSS 凭据、磁盘空间。

### Windows 上部分功能灰色

防火墙同步、FTP/邮件系统服务等依赖 Linux，Windows 仅适合开发演示。

### 旧链接 `/bastion`

已自动跳转到 `/terminal?tab=pam`，书签请更新。
